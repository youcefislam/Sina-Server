# Sina API Documentation

This is Sina' server documentation

## Description

A REST API which expose endpoints for [Sina](../../README.md#sina).

## Technologies

- NodeJs, expressJs, Jest, Joi, JWT, Sequelize, socketIO, twilio, sendGrid, moment, bcrypt, multer.
